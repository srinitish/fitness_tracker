# 🏋️ AI Fitness Planner using LangGraph & LLM

An AI-powered fitness planning system that automatically generates **personalized diet plans and workout schedules** using Large Language Models (LLMs).  
The system analyzes user inputs such as **age, weight, height, workout days, and fitness goal**, then calculates health metrics and generates structured recommendations.

This project demonstrates how **LLM workflows** can be built using **LangGraph** to orchestrate multiple AI steps.

---

## 🚀 Features

- 📊 **BMI and Calorie Calculation**
- 🥗 **AI-generated Personalized Diet Plan**
- 💪 **AI-generated Workout Schedule**
- 🧠 **Automated Fitness Summary**
- 🔄 **Multi-step AI workflow using LangGraph**
- ⚡ **Fast LLM inference using Groq API**
- 📦 **Structured JSON outputs**
